# React-demo
基于React-boostarp UI 样式
如果你是React初学者 这两个案例基本覆盖了React的所有基础，如果你喜欢就得star哦~
--------------------
使用：
下载项目地址
直接 npm install 或者用淘宝镜像cnpm 看你网速


留言板
支持在线发评论和记录当前 发布时间以及发布人
在鼠标移入到列表时，
删除按钮出现，
并且点击删除此条目。

##仿百度搜索下拉
点击搜索，获取到Jsonp 
重新渲染列表
方向盘 上下 
会改变当前条目的选中状态

Bug： 缺少选中条目时搜索框对应文字  以及选中的条目选中回车直接去百度页面的问题
